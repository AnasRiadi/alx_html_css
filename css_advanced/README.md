# Smileschool HTML & CSS Project

This project focuses on building a beautifully styled webpage for Smileschool using HTML and advanced CSS concepts. The goal is to create an aesthetically pleasing and fully responsive layout inspired by the provided Figma design.

## Features
- **Responsive navigation bar** for smooth user navigation.
- **Hero section** with a call-to-action button.
- **Instructor profiles** highlighting top contributors.
- **Testimonials** from satisfied learners.
- **Popular tutorials section** featuring smile-related lessons.
- **FAQ section** for common queries.
- **Footer with social media links**.

## Tools Used
- **HTML5** for structure.
- **CSS3** for styling and layout.
- **Font Awesome** for icons.
- **Source Sans Pro** and **Spin Cycle OT** fonts for typography.
